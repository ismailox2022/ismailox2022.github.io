Introduction à HTML

HTML (Hypertext Markup Language) est le langage de balisage standard pour la création de pages web et d'applications web. Il est utilisé pour structurer et formater le contenu sur le web et pour créer des liens entre différentes pages web. HTML se compose d'une série d'éléments, ou balises, qui sont utilisées pour définir la structure et le contenu d'une page web.

Balises HTML

Les balises HTML sont utilisées pour marquer le contenu d'une page web. Elles indiquent au navigateur comment interpréter et afficher le contenu. Voici quelques balises HTML courantes :

  • <p> : définit un paragraphe de texte
  • <h1> à <h6> : définissent des titres de différents niveaux
  • <a> : définit un lien hypertexte
  • <img> : définit une image
  • <div> : définit une division ou une section du document

Introduction à CSS

CSS (Cascading Style Sheets) est un langage de style utilisé pour décrire l'apparence d'un document HTML. Il permet de définir la mise en forme, la mise en page et l'apparence visuelle des éléments d'un site web. En utilisant CSS, vous pouvez contrôler la couleur, la police, l'espacement et de nombreux autres aspects de l'apparence de votre site.

Il existe plusieurs façons d'ajouter du CSS à un site web. Vous pouvez le faire directement dans l'élément <style> de votre page HTML, ou bien en utilisant un fichier CSS externe et en l'incluant dans votre page avec la balise <link>. Il est également possible d'utiliser des préprocesseurs CSS tels que SASS ou LESS pour ajouter des fonctionnalités avancées au CSS.

Propriétés CSS

Voici quelques propriétés CSS courantes et leur fonctionnalité :

  • color : cette propriété définit la couleur du texte d'un élément HTML. Elle peut être spécifiée en utilisant un nom de couleur comme "blue" ou en utilisant des valeurs de couleur en hexadécimal comme "#0000ff".
  • font-size : cette propriété définit la taille de la police de caractères pour un élément HTML. La taille peut être spécifiée en pixels (px), en points (pt) ou en em (un em est égal à la taille de la police actuelle). Par exemple, "20px" définira une taille de police de 20 pixels.
  • font-family : cette propriété définit la famille de polices à utiliser pour un élément HTML. Vous pouvez spécifier une seule police, comme "Arial", ou une liste de polices séparées par des virgules, comme "Arial, sans-serif". Si le navigateur ne dispose pas de la première police spécifiée, il passera à la suivante jusqu'à ce qu'une police disponible soit trouvée.
  • background-color : cette propriété définit la couleur d'arrière-plan d'un élément HTML. Elle peut être spécifiée en utilisant un nom de couleur comme "lightgray" ou en utilisant des v.

Introduction à JavaScript

JavaScript est un langage de programmation de scripts utilisé principalement dans les pages web interactives. Il permet de donner vie à un site en ajoutant des fonctionnalités telles que des pop-ups, des animations et des calculs dynamiques.

JavaScript peut être utilisé pour effectuer des actions lorsqu'un événement se produit sur une page, comme lorsqu'un utilisateur clique sur un bouton ou saisit du texte dans un formulaire. Vous pouvez également utiliser JavaScript pour effectuer des calculs et afficher le résultat sur votre page web.

Comme pour CSS, il existe plusieurs façons d'ajouter du JavaScript à une page web. Vous pouvez le faire directement dans l'élément <script> de votre page HTML, ou bien en utilisant un fichier JavaScript externe et en l'incluant dans votre page avec la balise <script>.

JavaScript est un langage de programmation de script côté client utilisé pour créer des sites web interactifs et dynamiques.

Fonctionnalités de JavaScript

  • Formulaires en ligne
  • Animations
  • Interactions avec l'utilisateur (clics, hover, etc.)
  • Communication avec les serveurs (Ajax, fetch, etc.)
